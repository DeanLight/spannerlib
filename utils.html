Args: level (logging.Level: optional): logging level to change the logger to. Defaults to logging.DEBUG. name (str: optional): module name to raise logging level for. Defaults to root logger toFile (Path: optional): File to output logs to. Defaults to None

Yields: [logging.Logger]: the logger object that we raised the level of*

run_cli_command

 run_cli_command (command:str, stderr:bool=False, shell:bool=False,
                  timeout:float=-1)

This utility can be used to run any cli command, and iterate over the output.

Type Default Details
command str a single command string
stderr bool False if true, suppress stderr output. default: False
shell bool False if true, spawn shell process (e.g. /bin/sh), which allows using system variables (e.g. $HOME),
but is considered a security risk (see: https://docs.python.org/3/library/subprocess.html#security-considerations)
timeout float -1 if positive, kill the process after timeout seconds. default: -1
Returns Iterable string iterator

 is_node_in_graphs (name, gs)
g = nx.Graph()
g.add_node(1)
g.add_node(2)
g.add_node('hello')

g2 = nx.Graph()
g2.add_node(1)
g2.add_node(2)
g2.add_node(3)
g2.add_node('hello_1')

assert _biggest_int_node_name(g) == 2
assert is_node_in_graphs(3,[g,g2])
assert not is_node_in_graphs(4,[g,g2])
assert get_new_node_name(g) == 3
assert get_new_node_name(g,'hello') == 'hello_0'
g.add_node('hello_0')
assert get_new_node_name(g,'hello') == 'hello_1'
assert get_new_node_name(g,'hello',avoid_names_from=[g2]) == 'hello_2'
assert get_new_node_name(g,'world') == 'world'
